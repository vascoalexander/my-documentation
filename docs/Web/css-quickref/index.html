@font-face​

Mit @font-face kannst du eine Schriftart direkt von einer URL (z. B. aus deinem Projekt oder einem CDN) laden und ihr einen Namen geben, der dann in CSS mit font-family verwendet werden kann.

@font-face { 
	font-family: 'Glegoo'; 
	src: url('../Glegoo.woff'); 
}
