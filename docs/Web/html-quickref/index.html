Image Mapsâ€‹

Image maps allow you to create interactive and clickable areas within an image to generate super features such as interactive diagrams, clickable infographics, or dynamic navigation menus. You can think of the map element as a special container that holds all your hotspots. Each hotspot is defined using the area element, which determines the shape and coordinates of the clickable area.
